Android端末のバッテリー情報を取得するプラグインです。

このプラグイン単独ではゲーム上で特に意味のある動作は行いません。

他のプラグインと組み合わせてうまいことご利用下さいませ。

使い方は、 js/plugins フォルダへ入れてプラグインマネージャーで登録するだけです。

詳しい処理は Battery.js を見て下さい。
