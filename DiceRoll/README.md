男爵さんオーダーメイドのプラグインです。
「DiceRoll('2d6')」とかやるとダイスっぽい乱数を出してくれます。

使い方は、 js/plugins フォルダへ入れてプラグインマネージャーで登録する。
そのあとは「プラグインコマンド」か「スクリプト実行」で呼び出して変数に入れて使って下さい。

詳しい処理は DiceRoll.js を見て下さい。
